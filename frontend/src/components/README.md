# Secure IoT Smart Energy Meter

A real-time IoT-based smart energy monitoring and billing dashboard developed using React, Node.js, and ESP32.

## Features

* Real-time energy monitoring
* Voltage and current analytics
* Smart billing estimation
* Daily and monthly usage tracking
* Interactive charts and analytics
* Device status monitoring
* Alert notification system
* Secure authentication system

## Technologies Used

* React.js
* Node.js
* Express.js
* Tailwind CSS
* ESP32
* Chart.js
* JavaScript

## Project Structure

* frontend → React frontend dashboard
* backend → Node.js backend APIs

## Future Enhancements

* MQTT integration
* Cloud deployment
* AI-based energy prediction
* Mobile application support
